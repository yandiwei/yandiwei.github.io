读书笔记_如何阅读一本书

发表于 2019-02-25 | 分类于 阅读

摘抄

太多的资讯就如同太少的资讯一样，都是对理解力的阻碍。

任何一种超越基础阅读的阅读层次，核心就在你要努力提出问题（然后尽你可能地找出答案）。

一个人如果说他知道他在想些什么，却说不出来，通常是他其实并不知道他在想什么。

所谓艺术或技巧，只属于那个能养成习惯，而且能依照规则来运作的人。

成人与孩子好奇心的区别——成人想要知道事情是否如此，而非为什么如此

阅读总讲

阅读目标

  • 为获得资讯而读——读者与作者的理解力在同一个层次
  • 为求得理解而读——读者与作者的理解力不在同一个层次

阅读艺术

这是一个凭借着头脑运作，除了玩味读物中的一些字句之外，不假任何外助，以一己之力来提升自我的过程。

四个层次

  • 基础阅读（elementary reading）——基本阅读——这个句子在说什么
  • 检视阅读（inspectional reading）——系统化略读——这本书在谈什么
  • 分析阅读（analytical reading）——完整阅读——咀嚼消化书
  • 主题阅读（syntopical reading）——比较阅读——同一个主题下的不同书

主动阅读

越主动，阅读的效果越好
一个主动的阅读者要提出的四个基本问题：

  1. 这本书整体上在讲什么？
  2. 作者细部说了什么？怎么说的？
  3. 这本书说的是否有道理？是全部有道理还是部分有道理？
  4. 这本书跟你有什么关系（对你有什么好处）？

阅读笔记

笔记不可或缺的原因

  • 保持清醒
  • 主动阅读是一种思考，思考倾向于用语言表达
  • 将感想写下来，能帮助记住作者的思想

三种笔记

  • 结构笔记（structural note-making）——全书的架构——检视阅读
  • 概念笔记（conceptural note-making）——与概念有关，作者的和自己的——分析阅读、主题阅读
  • 辩证笔记（dialectical note-making）——与主题有关的陈述与疑问——主题阅读

辅助阅读

按辅助来源分为四类

  • 相关经验
    分为一般经验和特殊经验。
经验来源 一般经验 特殊经验
小说 ✔
哲学 ✔
科学作品 ✔
历史 ✔ ✔
  • 其它的书
    其它的书可以当作阅读时的外在助力
  • 导论与摘要
    尽量少用，导论并不一定对，就算对也并不一定完整。在阅读全书之后再读，不然会被影响。
  • 工具书
    工具书包括字典和百科全书。
    明智使用工具书的四个条件:
    1. 知道想要找什么
    2. 知道用哪一种工具书
    3. 知道如何在工具书中找到答案
    4. 确定该书的编者或作者知道哪个答案

基础阅读

  • 阅读准备阶段——幼稚园
  • 认字——一年级
  • 字汇的增长及对课文的运用——四年级
  • 精练技巧——小学或初中

检视阅读

分为两步

  1. 有系统的略读或粗读——了解架构
  2. 粗浅的阅读——了解全书内容的第一个必要步骤

有系统的略读或粗读

建议：

  1. 先看书名页，然后有序就看序
  2. 研究目录页，对书的基本架构有概括性的理解
  3. 附有索引时大致看一下
  4. 读出版者的介绍
    ——以上四步判断是否想要更仔细地读下去还是放弃——
  5. 挑几个与主题有关的篇章看
  6. 随意看全书，留意主题的基本脉动

粗浅的阅读

头一次面对一本难读的书时，从头到尾先读完一遍，碰到不懂的地方不要停下来查询或思索。

阅读速度

理论上来说，不只是要能读得快，还要能在不同的地方用不同的速度，该快则快，该慢则慢。
影响阅读速度的原因：

  • 眼睛在移动时看不见，只有停下来时才能看见
  • 眼睛的焦点会逗留与倒退，影响阅读速度

提升阅读速度的方法：
用手指顺着一行一行字移动下去，速度比眼睛感觉快一点

分析阅读

第一阶段：找出一本书在谈些什么

  1. 依照书的种类与主题来分类
  2. 使用最简短的文字说明整本书在谈些什么
  3. 将主要部分按顺序与关联性列举出来，将全书的大纲列举出来，并将各个部分的大纲也列出来
  4. 确定作者想要解决的问题

第二阶段：诠释一本书的内容

  1. 诠释作者的关键字，与他达成共识（关键字理解的意思一致）
  2. 由最重要的句子中抓住作者的重要主旨
  3. 知道作者的论述是什么，从内容中找出相关的句子，再重新架构出来
  4. 确定作者已经解决了哪些问题，还有哪些是没解决的，再判断哪些是作者知道他没解决的问题

第三阶段：评论一本书

  1. 除非你已经完成大纲架构，也能诠释整本书了，否则不要轻易批评（在你说出‘我读懂了’之前，不要说你同意、不同意或暂缓评论）
  2. 不要争强好胜，非辩到底不可
  3. 在说出评论之前，你要能证明自己区别的出真正的知识和个人观点的不同

四种在对立角度的批评观点：

  • 作者的知识不足
  • 作者的知识错误
  • 作者不合逻辑
  • 作者的分析与理由不完整

对于前三种观点，无法提出相关佐证的，就必须同意作者说法或至少是一部分说法，对于第四种观点，只能由于这个理由才能对书暂缓评论。

主题阅读

准备阶段

  1. 针对要研究的主题设计一份试验性的书目
  2. 浏览书目上所有书，确定哪些与主题相关

阅读阶段

  1. 浏览上述被确定的书，找出最相关的章节
  2. 创造一套中立的词汇，和各个作者达成共识
  3. 建立一个中立的主题，列出问题
  4. 界定主要及次要议题
  5. 分析讨论

阅读不同读物

基本用上述所学，结合主动阅读者的四个问题再加上一些技巧。

  • 实用性的书——一定要行动
  • 想像文学——打开心灵接纳它
  • 故事书——快读、全心全意地读
  • 戏剧——自己提供身体语言的实际演出
  • 抒情诗——不论懂不懂，一口气读完；大声重读一遍，多了解作者及背景资料
  • 历史书——阅读一种以上相关历史书；不要只关心过去发生了什么，还要想想现在还发生这种事情的原因
  • 科学与数学——目的是了解相关问题而不是成为这个主题的专家，这样会轻松一点
  • 哲学书——找中心思想，不断思考
  • 社会科学——主题阅读
