变量

作用

变量是用来储存值的

命名规则

  • 变量名只能包含字母、数字和下划线。变量名可以字母或下划线打头,但不能以数字打头。
  • 变量名不能包含空格,但可使用下划线来分隔其中的单词。
  • 不要将 Python 关键字和函数名用作变量名
  • 变量名应既简短又具有描述性。
  • 慎用小写字母 l 和大写字母 O ,因为它们可能被人错看成数字 1 和 0 。

数据类型

字符串

定义

字符串就是一系列字符，并且用单引号或双引号括起。

操作

字符串大小写

首字母大写：字符串变量名.title();
字符串全大写：字符串变量名.upper();
字符串全小写：字符串变量名.lower()。
如果name = "HELLO world",那么
name.title() = "HELLO World";
name.upper() = "HELLO WORLD";
name.lower() = "hello world"。

合并字符串

+
如果A = "Hello", B = "World",那么
A + B = "HelloWorld"
A + " " + B = "Hello World"

添加空白

空白泛指任何非打印字符,如空格、制表符和换行符。
在字符串中添加制表符：\t;
在字符串中添加换行符：\n;

删除空白

删除字符串两边空白：字符串变量名.strip();
删除字符串左边空白：字符串变量名.lstrip();
删除字符串右边空白：字符串变量名.rstrip()。
如果name = " python ",那么
name.strip = “python”;
name.lstrip = “python ”;
name.rstrip = “ python”。

列表

定义

列表由一系列按特定顺序排列的元素组成。

表示

在 Python 中,用方括号[]来表示列表,并用逗号来分隔其中的元素。
例如：me = ['Yan Diwei', 'Male', '26'](下文me结尾该值)。

访问列表元素

  • 索引从0开始，即第一个元素排序为0,第二个为1,之后2,3,……;
  • 特殊语法：索引指定为-1,可让Python返回最后一个列表元素，-2则是倒数第二个，依此类推。

例如
me[0] = 'Yan Diwei';
me[-3] = 'Yan Diwei'。

修改列表元素

直接赋予新值即可。
输入me[0] = 'Diwei Yan'，那么
me = ['Diwei Yan', 'Male', '26']。

在列表中添加元素

在列表末尾添加元素

列表变量名.append()
输入me.append('Shaoxing')，那么
me = ['Yan Diwei', 'Male', '26', 'Shaoxing']。

在列表中插入元素

列表变量名.insert(索引, 元素)
输入me.insert(2, 'Shaoxing')，那么
me = ['Yan Diwei', 'Male', 'Shaoxing', '26']。

从列表中删除元素

根据索引号删除

del 列表变量名[索引号]
列表变量名.pop(索引号)，索引号缺省则为最末尾元素。
区别：pop()返回值，del不返回，相当于Pascal中的函数与过程的区别，从格式上可以看出。

输入del me[1],那么
me = ['Yan Diwei', '26'];
输入me.pop（1），那么
me = ['Yan Diwei', '26']。

根据值删除

列表变量名.remove(元素值)
输入me.remove(Male),那么
me = ['Yan Diwei', '26'];
注意：remove.()只删除第一个值，如果删除的值有多个，则要用到循环。

列表变量名.sort()
