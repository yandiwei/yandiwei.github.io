Markdown与HTML的关系

  • Markdown 是一种书写的格式，HTML 是一种发布的格式。
  • Markdown只对应HTML标记的一小部分，HTML不在 Markdown 涵盖范围之内的标签，都可以直接在文档里面用 HTML 撰写。
  • HTML 文件中， < 和 & 需要特殊处理成&lt和&amp，Markdown自动转换。
